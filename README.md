# 第二大脑 · Second Brain

一个基于 Next.js 的个人知识管理系统，用于复盘笔记、对话记录和记忆碎片。

## 🎯 功能特性

### ✅ 已完成
- **全量列表展示** - 支持搜索的记忆点列表，按时间倒序排列
- **全局搜索 (Cmd+K)** - 快捷键唤起的全局搜索，覆盖所有内容类型
- **多维筛选** - 按内容类型、日期范围、标签进行过滤
- **极致简约 UI** - Minimal Design 设计风格，专注内容本身
- **响应式设计** - 适配桌面和移动设备
- **类型统计** - 可视化展示各类记忆的分布情况
- **模拟数据** - 包含示例数据，便于演示和测试

## 🚀 快速开始

### 环境要求
- Node.js 18+ 
- npm 或 yarn

### 安装依赖
```bash
npm install
```

### 启动开发服务器
```bash
npm run dev
```

访问 http://localhost:3000 查看应用。

### 构建生产版本
```bash
npm run build
npm start
```

```
second-brain/
├── app/                    # Next.js App Router
```

## 🎨 设计理念

### Minimal UI 原则
- **极简布局** - 减少视觉噪音，专注内容
- **充足留白** - 提升可读性和呼吸感
- **一致间距** - 使用 Tailwind 的间距系统
- **克制用色** - 主色调+中性色，突出重点内容

### 用户体验
- **快捷键优先** - Cmd+K 全局搜索，Esc 关闭
- **渐进式交互** - 列表项点击展开详情
- **实时反馈** - 搜索和筛选即时响应
- **移动友好** - 触控友好的交互设计

## 🔧 技术栈

- **Next.js 14** - React 框架，App Router
- **TypeScript** - 类型安全
- **Tailwind CSS** - 实用优先的 CSS 框架
- **Lucide React** - 简洁的图标库
- **React Hooks** - 状态管理

## 📱 核心功能详解

### 全局搜索 (Cmd+K)
- 支持标题、内容、标签的模糊搜索
- 实时显示搜索结果
- 键盘导航支持（↑↓ 箭头，Enter 选择）
- 搜索结果高亮显示

### 多维筛选
1. **内容类型** - 笔记、对话、任务、想法、书签、其他
2. **时间范围** - 今天、本周、本月、今年、自定义
3. **标签筛选** - 多标签组合筛选

### 记忆列表
- 卡片式布局，每项显示关键信息
- 点击展开查看完整详情
- 按创建时间倒序排列
- 支持分页加载

## 🔌 数据集成

当前使用模拟数据，位于 `lib/data.ts`。要连接真实数据源：

1. 修改 `lib/data.ts` 中的函数，连接你的数据库
2. 或创建 API 路由，从后端服务获取数据
3. 支持的数据源：PostgreSQL、MySQL、SQLite、MongoDB 等
