# 第二大脑 - ReVibe项目知识库

## 🎯 项目概述

**ReVibe** 是一个基于循证运动科学的ACL术后运动能力重建系统，通过阶段化训练指导、视频动作评估和个性化训练计划，帮助用户安全恢复运动能力。
